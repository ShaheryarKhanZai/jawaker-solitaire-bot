# Contributing

## What's Welcome

- Bug fixes
- Improving card detection accuracy
- Support for other screen resolutions
- Performance improvements
- Better solver logic (more move strategies)

## How to Contribute

1. Fork the repo
2. Create a branch: `git checkout -b fix/your-fix-name`
3. Make your changes
4. Test that the bot still runs and calibrates correctly
5. Commit with a clear message: `git commit -m "fix: describe what you changed"`
6. Push and open a Pull Request

## Code Style

- Keep functions focused — one job per function
- Add a comment if the logic isn't obvious
- Don't break the existing GUI layout without good reason

## Reporting Bugs

Open a GitHub Issue with:
- Your OS and Python version
- Which bot variant you're running
- What you expected vs what happened
- Any error messages from the terminal
